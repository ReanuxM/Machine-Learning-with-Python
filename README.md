# 🚗 CO2 Emission Prediction using Multiple Linear Regression

## 📌 Project Overview
This project applies Machine Learning techniques to predict vehicle CO2 emissions based on engine size and fuel consumption using Multiple Linear Regression.

The dataset used is the Fuel Consumption dataset from IBM Developer Skills Network.

---

## 🧠 Objectives
- Understand relationships between vehicle features and CO2 emissions
- Apply data preprocessing and feature scaling
- Build and evaluate a Multiple Linear Regression model
- Visualize results in both 2D and 3D

---

## 📊 Dataset
The dataset includes:
- Engine Size
- Fuel Consumption
- CO2 Emissions

Source:
IBM Developer Skills Network

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🧪 Workflow
1. Load dataset
2. Clean and preprocess data
3. Feature scaling using StandardScaler
4. Split data into training and testing sets
5. Train Multiple Linear Regression model
6. Evaluate and visualize results

---

## 📈 Results
- Model successfully predicts CO2 emissions
- Engine size and fuel consumption are strong predictors
- Visualization shows clear linear relationship in 3D space

---

## 📊 Visualizations
- Correlation heatmap
- 3D regression plane
- Scatter plots with regression lines
